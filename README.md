# J.B. Hunt Transport Services (jb-hunt-transport-services)

J.B. Hunt Transport Services, Inc. is one of the largest transportation and logistics companies in North America, providing trucking, intermodal, and contract services. J.B. Hunt offers the J.B. Hunt 360 Connect API portal for shippers and partners to programmatically manage freight operations.

## APIs

### J.B. Hunt 360 Connect API

Comprehensive API for managing freight operations on the J.B. Hunt 360 platform. Supports full-truckload (FTL) and less-than-truckload (LTL) shipments.

**Key capabilities:**

- **Quotes** - Request on-demand quotes for FTL and LTL shipments at market or contractual rates
- **Order Management** - Create, search, look up, and modify transportation orders
- **Tracking** - Near real-time freight shipment tracking
- **Document Management** - Upload and retrieve shipment documents (bills of lading, proof of delivery)
- **Scheduling** - Set and manage pickup and delivery appointments

**Base URL:** `https://api.jbhunt.com`
